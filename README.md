## О проекте

Одностраничный лендинг образовательного продукта с акцентом на конверсию: понятная структура, интерактивные блоки и простая форма регистрации.

### Целевая аудитория

- **Основная:** студенты и начинающие специалисты 18–35 лет, которые хотят освоить UX/UI и собрать портфолио
- **Дополнительная:** работающие специалисты 25–40 лет, желающие переквалифицироваться

### Реализованные разделы

| Блок | Описание |
|------|----------|
| Header | Фиксированная шапка, якорная навигация, бургер-меню |
| Hero | Заголовок, CTA, статистика, иллюстрация |
| Преимущества | 4 карточки с hover-анимациями |
| Программа | Аккордеон из 8 модулей |
| Отзывы | Карточки выпускников и блок партнёров |
| Регистрация | Форма с JS-валидацией |
| Footer | Контакты, соцсети, юридические ссылки |

### Фирменный стиль

## Технологии

- HTML5 (семантическая разметка)
- CSS3 (адаптивная вёрстка, CSS-переменные, анимации)
- JavaScript (бургер-меню, аккордеон, валидация формы, scroll reveal)

Без фреймворков и сборщиков — статический сайт, готовый к деплою на GitHub Pages.

## Структура репозитория

```
.
├── index.html          # Готовая страница (стили встроены)
├── style.css           # Черновая wireframe-вёрстка (этап прототипирования)
├── images/
│   └── logo.png        # Логотип проекта
├── docs/
│   └── Румянцева_Е.И..docx   # Отчёт по курсовой работе
└── README.md
```

## Локальный запуск

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере

Или через локальный сервер:

```bash
npx serve .
```

## JavaScript-функциональность

- Плавная прокрутка по якорям
- Бургер-меню для мобильных устройств
- Аккордеон программы (один модуль открыт одновременно)
- Валидация формы регистрации (имя + email)
- Анимации появления блоков при скролле
- Hover-эффекты на карточках

## Адаптивность

Вёрстка протестирована для:

- Desktop (≥1024px)
- Tablet (768–1024px)
- Mobile (≤768px)
